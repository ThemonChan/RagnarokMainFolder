[ R A G N A R O K o n l i n e : skin making manual ] 라그나로크의 GUI(Graphic User Interface)는 스킨 시스템을 적용하여 제작되었기때문에 몇가지 방법만 잘 숙지하면 유저가 직접 제작하거나 변경할 수 있습니다. 이렇게 제작된 스킨의 경우, 로그인 인터페이스를 제외하고는 모두 해당 스킨으로 변경되어 게임에 나타나게 됩니다. 0. 추가 갱신 사항 - 2002년 5월 31일자 길드 인터페이스 추가에 따라, 길드 로고를 변경하도록 하는 btn_edit 버튼, 길드원에게 전체 메세지를 발송하는 btn_send 버튼, 카드 그림을 볼 수 있는 btn_view 버튼이 추가되 었습니다. 1. 시작하기 먼저 라그나로크 기본 스킨을 포함하고 있는 ragnarok_basic.zip 파일을 다운로드합니다. 해당 압축파일을 풀어서 작업을 진행할 디렉토리에 저장합니다. 스킨은 해당 폴더의 루트에 있는 74개의 파일과 basic_interface라는 서브 폴더에 있는 124개의 파일을 수정하는 작업을 통해 제작됩니다. 되도록 확인하면서 작업할 수 있게 라그나로크가 설치된 폴더안에 skin이란 폴더를 만들고 (예 : c:\program files\gravity\ragnarok\skin) 그 안에 해당 스킨명을 갖는 폴더를 만들어 풀어주는 것이 좋습니다. 포토샵이나 페인트샵 등, 픽셀단위의 편집이 가능한 그래픽 프로그램을 준비합니다. 2. 작업하기 가장먼저 주의할 점은, 게임에 있어서 투명하게 표현할 부분의 색깔은 R255, G0, B255의 색이라는 것입니다. 색이 조금이라도 다를경우 투명하게 표현되지 않을 수 있으므로, 꼭 투명색은 RGB(255, 0, 255)라는 점을 잊지 말아주세요! 파일은 반드시 bmp로 저장되어야하며, 포토샵의 image>mode>index color 메뉴를 통해 256 칼라로 변환해주면 용량을 아낄 수 있습니다. 또!! 텍스트, 버튼, 그래프등은 정해진 위치에만 나타나므로 주의해서 디자인해야합니다. (*로 표시된 부분은 많은 수의 이름을 대신해 표현한 것임.) 이름에 '_dis' 가 붙는 것은 해당 버튼이 사용될 수 없을때 btn_resize.bmp : 리사이즈가 가능한 창의 우측하단에 나타나는 리사이즈 버튼 checkbox_*.bmp : 옵션이나 오늘의 팁등에 사용되는 체크박스의 그림 이름에 '_0' 이 붙는 것은 비활성화된 체크박스 이름에 '_1' 이 붙는 것은 활성화된 체크박스 colorchip.bmp : 게임 내에서 프로그램이 직접 출력하는 부분, 순서는 다음과 같음 칼라칩은 32*32의 크기이며, 각각 4*4의 칩으로 이루어짐. 이 칩을 각각 좌측상단부터 한줄씩 우측으로 1,2,3,4 / 5,6,7,8 으로 순서 있음 칼라칩은 반드시 index color을 해서 256 칼라로 변환해 저장해야함!! 각 괄호안의 숫자는 (Red, Green, Blue) skin 폴더를 만들고 그 안에 자신이 만드는 스킨의 이름을 갖는 폴더를 만들어 작업을 하며 지속적으로 게임에 들어가 확인하면서 하는 것이 좋습니다. 4. 주의사항 스킨 폴더는 라그나로크가 설치된 폴더에 있어야합니다. bgm이나 screenshot등과 같은 위치에 있어야 한다는 것이죠. 스킨 폴더 하위에 있는 폴더나 winzip포맷의 압축파일(alzip포맷은 제대로 지원되지 않을 수도 있습니다.)을 하나의 스킨으로 인식합니다.
